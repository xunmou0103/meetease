## [1.2.0] - 2026-05-16

### Added
- **用户登录**：独立登录页 `/login`，未登录访问业务页自动跳转
- 支持 7 种主流账户模拟登录：微信、钉钉、飞书、企业微信、手机号、Google、GitHub
- **首次登录自动注册** MeetEase 助手账户（生成 `assistantId`，写入 localStorage）
- 顶栏用户菜单：展示昵称、邮箱、助手 ID、登录渠道、退出登录
- 新用户登录后工作台欢迎 Toast
- 待办页「我是」默认关联登录用户（若与演示负责人重名）

### 文档
- PRD 升级至 v1.2.0，新增登录与账户模块（AUTH-*）

## [1.1.0] - 2026-05-16

### Changed
- **「会议记录」更名为「AI 会议分析」**（路由 `/analysis`，`/record` 自动重定向）
- 导航、工作台快捷入口、接入引导文案同步更新

### Added
- AI 会议分析页 **选择会议** 下拉框，支持多场会议（含「进行中」状态）
- 各会议独立转写文本，切换会议时加载对应内容
- 按会议 ID 差异化的 AI 分析预设结果（m1/m2/m3）

### Removed
- 页面组件 `MeetingRecord.jsx`（由 `MeetingAnalysis.jsx` 替代）

### 文档
- PRD 升级至 v1.1.0，需求编号 REC-* 调整为 ANA-*

---

## [1.0.0] - 2026-05-16

### Added
- **工作台**：今日会议/待办统计、接入引导首屏卡片、快捷入口、最近纪要、高优先级待办
- **接入会议**：三种接入方式说明；腾讯会议、钉钉、飞书、企业微信（内测）、Zoom 分步引导；模拟完成接入（localStorage）
- **创建会议**：主题/时间/参会人/目标/背景资料表单；AI 生成议程、讨论重点、准备清单
- **会议记录**：转写文本编辑；提取会议重点、识别待办、生成摘要
- **会议纪要**：简洁版/正式版/项目管理版（内容差异化）；一键复制；生成会后邮件
- **待办事项**：表格管理；状态筛选；负责人「我的待办」筛选；状态切换；负责人 localStorage 记忆
- **全局**：顶栏导航（接入会议置前高亮）、PageHeader 场景徽章、产品测评 Demo 标签、页脚声明
- **视觉**：Plus Jakarta Sans、蓝紫 Design Token、卡片化布局、响应式适配

### Changed
- 接入会议强调色由青绿色调整为靛蓝紫渐变，与主品牌色系统一
- 工作台接入引导移至统计卡片上方

### 文档
- 首版 PRD（v1.0.0）与文档维护规范发布
