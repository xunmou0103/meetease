# MeetEase 产品文档

| 文档 | 说明 |
|------|------|
| [PRD.md](./PRD.md) | **产品需求文档**（功能范围、验收标准、用户旅程） |
| [CHANGELOG.md](./CHANGELOG.md) | **产品变更日志**（按版本记录功能增删改） |

## 维护流程（必读）

每次更新功能时，请按顺序完成：

1. 修改代码并实现功能  
2. 更新 `CHANGELOG.md`（在 `[Unreleased]` 或新版本下记录）  
3. 更新 `PRD.md` 对应章节、验收标准，并递增文档版本号  
4. 若影响使用方式，同步更新根目录 `README.md`  

详细规范见 [PRD §12 文档维护规范](./PRD.md#12-文档维护规范)。
